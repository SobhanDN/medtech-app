اتصال به Google Sheets
داده‌های شما در یه Google Sheet ذخیره میشن. هر دو نفر با باز کردن همین فایل HTML به داده‌های مشترک دسترسی دارن.
۱
به sheets.google.com برو و یه Sheet جدید بساز. اسمش مثلاً «حسابداری مدیریت» بذار.
۲
از منو Extensions → Apps Script رو باز کن، کد زیر رو paste کن و Save کن، بعد Deploy → New deployment رو بزن (نوع: Web app، دسترسی: Anyone).

۳
